# 압구정곱창 메뉴얼 - 데이터베이스

## 폴더 구조

```
압구정곱창_메뉴얼/
├── _raw/                          # 원본 파일 저장소
│   └── 원본_입력폴더/             # 기입한 원본 파일들을 여기에 넣으세요
├── db_schema.json                 # 메뉴 DB 구조 정의
├── menu_categories.json           # 메뉴 카테고리 분류
├── menu_items.json                # 메뉴 항목 상세
├── menu_options.json              # 추가옵션 (부위, 세트 등)
├── menu_analysis.md               # 분석 결과 및 정규화
└── README.md                      # 이 파일
```

## 사용 방법

1. **원본 입력**: `_raw/원본_입력폴더/` 에 기입한 메뉴 원본 파일 추가
2. **분석**: DB 구조에 따라 JSON 파일들 정규화
3. **메뉴얼**: 정규화된 데이터로 메뉴얼 문서 생성

## DB 구조

### 카테고리 (Categories)
- 곱창류
- 소곱창류
- 육류
- 음료
- 반찬/사이드
- 세트메뉴

### 메뉴 항목 (Menu Items)
- 메뉴 ID
- 메뉴명
- 카테고리
- 가격
- 설명
- 특성 (매운맛, 추천 등)

### 옵션 (Options)
- 부위 선택
- 분량 선택
- 세트 구성
- 추가 요금
