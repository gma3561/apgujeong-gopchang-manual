🔥 그릴 조리 프로세스
1단계 - 준비
중량에 맞게 계량 → 하단 불판 올리기
강불로 착화

2단계 - 초벌 구우기 (약 1분 30초)
• 곱창: 기름이 눈에 보이도록 고기부분이 불판에 닿게 배치
• 대창: 중량에 맞게 불판 중간 아래 배치
• 막창: 맨밑에 배치
• 기름이 튀기시작 → 큰불 줄이고 가운데불로만 조리

3단계 - 색 조절 & 재배치
• 색이 많이 나는 곱창 → 불판 맨끝 또는 모서리로 이동
• 얇은 곡창 → 덜구워진면을 불판에 닿게 배치
• 대창 노랗게 변하면 → 곱창 아래 우측에 배치
• 막창 구워지기시작 → 곱창 아래 좌측에 배치
• 염통 → 가운데에 배치

4단계 - 야채 배치
감자 2개 / 단호박 2개 / 양파 50g / 대파 50g / 새송이 50g
떡 4개 (정갈하게 배치)

4단계 - 고객 응대

5단계 - 소스 권장

막창 → 기름장
