## 👨‍💼 직원별 상세 업무

### 1️⃣ 봉녀 (고기 그릴러)

**포지션**: 메인 그릴 담당  
**근무시간**: 오전 준비 ~ 마감  
**주요 책임**: 고기 품질 & 조리 타이밍

| 시간 | 업무 |
|------|------|
| **아침** | 고기 정리 (곱창/대창/막창/양/염통/간) |
| | 일일 필요량 계산: 평일 4-5봉, 주말 6-7봉 |
| | 각 고기를 쟁반/바트에 정렬, 숙성고 준비 |
| **영업중** | 주문 받으면 실시간 고기 구우기 |
| | 불판 배치: 곱창 상단, 대창 중간, 막창 하단 |
| | 타이밍: 약 1분 30초 기준 |
| | 색 조절: 타는 부분은 모서리로 이동 |
| **마감** | 고기 정리 및 발주 (다음날) |

**⚠️ 체크포인트**:
- 일일 고기량 정확한 계산
- 고기 신선도 매일 확인
- 불 세기 조절 능력

## 🍖 레시피 & 포션

### 📌 계란찜

**수량**: 1인분 (1호 뚝배기)  
**담당**: 옥화

| 재료 | 수량 | 준비 |
|------|------|------|
| 계란물 | 1.5국자 | 계란 깨서 물+맛소금 추가, 홍두깨 그라인더로 간 것 |
| 계란찜 육수 | 1국자 | 멸치+디포리+소금으로 끓인 후 한소끔 식힌 것 |

**조리 과정**:
1. 뚝배기에 계란물 1.5 + 육수 1국자 혼합
2. 강불에서 뜨거울 때까지 올림
3. 뚝배기 옆부분 익으면 약불로 줄이기
4. 뚜껑 덮고 속까지 익히기
5. 연기 나면 불 끄기
6. 삼바리에 받쳐 제공

---

### 🔥 구이 (1인분)

**담당**: 봉녀  
**기본 정량**:

| 부위 | 수량 | 준비 방법 |
|------|------|----------|
| 곱창 | 160g | 5cm 간격, 기름/못난이 제거 후 쟁반에 편으로 나열, 숙성고 준비 |
| 대창 | 230g | 5cm 간격, 기름/못난이 제거 후 1/2 바트에 키친타올 깔고 준비 |
| 막창 | 150g | 편으로 썬 후 1/2 바트에 키친타올 깔고 준비 |
| 양 | 150g | 진공포장 뜯어서 키친타올로 수분 제거 후 준비 |

**가니쉬 (한 판 기준)**:
- 부추무침: 100-120g (부추 양념에 버무린 것)
- 양파: 50g
- 대파: 50g
- 호박: 30g
- 감자: 2조각 (약 60g, 끓는 물에 데친 후)
- 새송이: 30g (0.5cm 간격 슬라이싱)
- 떡: 4개

**조리 타이밍**: ~1분 30초

## 📋 매뉴얼 부착 위치 (인쇄 시 참고)

- **내용**: 계란찜 레시피, 구이 정량, 가니쉬 세팅, 전골 구성

1. **모든 수치는 기록이 생명**

   - 야채: 매일 아침 신선도 확인
